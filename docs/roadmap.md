# BaseYield Roadmap

## Phase 1 - Thesis Demo and Prototype (Current)

- Publish repository with clear product framing
- Launch landing page and simulated dashboard
- Define mode-specific product profiles:
  - Money Mode
  - Yield Mode
- Scaffold Base-compatible smart contracts

## Phase 2 - Testnet Vault Simulation

- Deploy prototype contracts to Base Sepolia
- Connect dashboard to testnet vault state
- Add event-driven accounting updates
- Introduce scenario simulator for reserve and redemption behavior

## Phase 3 - Integration Exploration

- Evaluate offchain integration partners for custody and reporting
- Explore pricing and reconciliation pipelines
- Define operational controls for liquidity windows and buffers
- Expand monitoring and transparency framework

## Phase 4 - Regulated Wrapper Exploration

- Evaluate legal structures for credit-linked wrappers
- Design compliance-aware distribution model
- Implement stronger risk governance and disclosure standards
- Prepare path toward production architecture

## Continuous Workstreams

- Smart contract hardening and testing depth
- Product UX refinement for clarity around risk and liquidity
- Community and ecosystem collaboration around digital credit primitives
